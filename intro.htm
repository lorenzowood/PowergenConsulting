Powergen group technology consulting project

15th October 2001

Summary

In this short project, where we looked at how to get the Powergen Group working more effectively, we did the following:

  • Interviewed the Group directors with a couple of exceptions, together with some of their PAs. The raw output of this exercise can be found under Interviews.
  • Derived a set of broad, common issues arising from these interviews. The issues can be found under Issues, where you can also see from the breakdown what evidence we had for these issues from the interviews.
  • We then produced an initial set of ideas for things to address some of the issues and to improve working practices in the group. These can be found under ideas.
  • To give the project momentum, we needed to put these ideas into a framework that the organisation can use as a handle on the project going forward. We created Super connectivity to do this. Super connectivity is set of principles that can be used both as a diagnostic and for planning purposes. In the ideas section, we have arranged the ideas under each of the principles for reference.
  • Finally, we applied our process for Building value in a people business to the Powergen group, where we see that:

    We have begun successfully by engaging the most senior level of the group. To keep the momentum, we build on their stated wish to work effectively and collaboratively by providing them with individual benefits (the “selfish participant” part of our process), and then plan to move forward by first strongly supporting this change in working practices, and then by measuring it as well. In 12 months’ time, we suggest that (at least) directors’s objectives should include targets for collaborative working.

This Web site contains the core output of this project. At the start of the project, we created a small kick-off site to introduce it. We have preserved this here.
